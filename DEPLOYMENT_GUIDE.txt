=================================================================
   TROUBLESHOOTING
=================================================================

Issue: "git pull" fails with local changes
Solution:
  git stash
  git pull origin master
  git stash pop

Issue: "npm run build" fails
Solution:
  rm -rf node_modules
  npm install
  npm run build

Issue: Lists still not showing
Check:
  1. Browser console for errors (F12)
  2. Network tab - check API responses
  3. Server error logs

Issue: Permission denied
Solution:
  chmod 755 frontend/dist
  chmod 644 frontend/dist/*
